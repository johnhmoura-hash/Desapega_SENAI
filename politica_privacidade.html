Política de Privacidade

Coleta de dados

Coletamos apenas os dados necessários para o funcionamento do site.

Informações armazenadas

  • Nome;
  • E-mail;
  • Foto de perfil (opcional);
  • Anúncios publicados.

Uso das informações

Os dados são utilizados apenas para permitir o funcionamento do sistema e facilitar as negociações.

Compartilhamento

O Desapega SENAI não vende nem compartilha informações pessoais com terceiros.

Segurança

Adotamos medidas para proteger os dados dos usuários contra acessos não autorizados.
